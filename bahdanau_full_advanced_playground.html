Step 1 — alignment score (additive)
e(s,t) = vᵀ · tanh( W₁·hₛ + W₂·hₜ₋₁ )
hₛ = encoder hidden state at position s (what the encoder "remembers" up to word s)
hₜ₋₁ = previous decoder hidden state (what the decoder "wants" to say next)
W₁, W₂, v = learned weight matrices — trained via backprop
tanh = squishes values to [-1,+1] so gradients flow well
Step 2 — softmax normalization
α(s,t) = exp( e(s,t) ) / Σₛ exp( e(s,t) )
Converts raw scores into a probability distribution. All α values sum to 1.
High α(s,t) → decoder pays more attention to encoder position s when generating output word t.
Step 3 — context vector
c(t) = Σₛ α(s,t) · hₛ
Weighted sum of ALL encoder hidden states. Think of it as a spotlight — α weights control how bright each encoder state shines into the context.
This vector changes at every decoder step, unlike vanilla seq2seq where it's fixed.
Step 4 — decoder output
yₜ = softmax( Wₛ · concat(hₜ , c(t)) + b )
The decoder concatenates its own hidden state hₜ with the context vector c(t), then projects to vocabulary size to get a probability over all possible next words.

Vanilla seq2seq 2014
h_final → single context vector
+ Simple, fast to train
+ Works well for short sentences (<30 words)
— Single vector bottleneck
— Forgets early words in long sequences
— BLEU score drops sharply after 30 tokens
Recommended
Bahdanau attention 2015
dynamic c(t) = Σ α(s,t)·hₛ
+ Dynamic context per decoder step
+ Handles long sequences well
+ Interpretable attention maps
+ Significant BLEU improvement
— Quadratic complexity O(n×m)
— Still sequential (LSTM dependency)
Luong attention 2015
score = hₜᵀ · Wₐ · hₛ
+ Simpler multiplicative score
+ Slightly faster computation
+ Attends AFTER decoding step
— Less flexible than additive
— Works less well on very different length sequences
Transformer (self-attention) 2017
score = Q·Kᵀ / √dₖ
+ No LSTM — fully parallel
+ Multi-head attention
+ Powers BERT, GPT, ChatGPT
+ O(n²) but GPU-parallelisable
— Needs huge data + compute

PyTorch — Bahdanau attention
import torch, torch.nn as nn, torch.nn.functional as F

class BahdanauAttention(nn.Module):
    """Additive (concat) attention — Bahdanau 2015"""
    def __init__(self, enc_hidden, dec_hidden):
        super().__init__()
        self.W1  = nn.Linear(enc_hidden, dec_hidden, bias=False)
        self.W2  = nn.Linear(dec_hidden, dec_hidden, bias=False)
        self.v   = nn.Linear(dec_hidden, 1,          bias=False)

    def forward(self, encoder_states, decoder_hidden):
        # encoder_states : [B, src_len, enc_hidden]
        # decoder_hidden : [B, dec_hidden]
        h_t = decoder_hidden.unsqueeze(1)      # [B, 1, dec_hidden]
        energy = torch.tanh(
            self.W1(encoder_states)            # [B, src, dec_h]
          + self.W2(h_t)                       # broadcast
        )                                      # [B, src, dec_h]
        scores  = self.v(energy).squeeze(-1)   # [B, src_len]
        alpha   = F.softmax(scores, dim=-1)    # [B, src_len]
        context = (alpha.unsqueeze(2)          # [B, src, 1]
                   * encoder_states).sum(1)    # [B, enc_h]
        return context, alpha                  # context + weights
TensorFlow / Keras version
import tensorflow as tf

class BahdanauAttention(tf.keras.layers.Layer):
    def __init__(self, units):
        super().__init__()
        self.W1 = tf.keras.layers.Dense(units)
        self.W2 = tf.keras.layers.Dense(units)
        self.V  = tf.keras.layers.Dense(1)

    def call(self, query, values):
        # query  : [B, hidden]   decoder state
        # values : [B, seq, h]   encoder states
        q_exp  = tf.expand_dims(query, 1)      # [B, 1, h]
        score  = self.V(tf.nn.tanh(
            self.W1(values) + self.W2(q_exp))) # [B, seq, 1]
        alpha  = tf.nn.softmax(score, axis=1)  # [B, seq, 1]
        context = tf.reduce_sum(
            alpha * values, axis=1)            # [B, hidden]
        return context, alpha
Key formulas
Alignment: e(s,t) = vᵀ · tanh(W₁hₛ + W₂hₜ₋₁)
Weights: α(s,t) = softmax(e(s,t))
Context: c(t) = Σₛ α(s,t) · hₛ
Output: yₜ = softmax(Wₛ·[hₜ ; c(t)] + b)
