What is attention?
In vanilla seq2seq (encoder-decoder), the entire input sentence gets squeezed into a single fixed-size context vector. For long sentences this bottle-necks — early words get forgotten.

Attention solves this by giving the decoder dynamic access to ALL encoder hidden states at every decoding step, weighted by relevance.
Bahdanau vs Luong attention
Bahdanau (additive): score = vᵀ · tanh(W₁·hₛ + W₂·hₜ)
Uses a feedforward network. Attends BEFORE decoding current word.

Luong (multiplicative): score = hₜᵀ · Wₐ · hₛ
Simpler dot-product style. Attends AFTER decoding current word.

PyTorch — Bahdanau attention (additive)
import torch
import torch.nn as nn
import torch.nn.functional as F

class BahdanauAttention(nn.Module):
    def __init__(self, hidden_size):
        super().__init__()
        self.W1 = nn.Linear(hidden_size, hidden_size)
        self.W2 = nn.Linear(hidden_size, hidden_size)
        self.v  = nn.Linear(hidden_size, 1)

    def forward(self, query, keys):
        # query: [batch, hidden]  (decoder hidden state)
        # keys:  [batch, seq_len, hidden]  (all encoder hidden states)
        query = query.unsqueeze(1)          # [batch, 1, hidden]
        energy = torch.tanh(
            self.W1(keys) + self.W2(query)  # broadcast add
        )                                   # [batch, seq_len, hidden]
        scores  = self.v(energy).squeeze(-1)# [batch, seq_len]
        weights = F.softmax(scores, dim=1)  # [batch, seq_len]
        # Weighted sum of encoder states
        context = torch.bmm(
            weights.unsqueeze(1), keys      # [batch, 1, hidden]
        ).squeeze(1)
        return context, weights
Alignment score formula
e(s,t) = vᵀ · tanh( W₁·hₛ + W₂·hₜ₋₁ )
α(s,t) = softmax( e(s,t) ) ← attention weights
c(t) = Σₛ α(s,t) · hₛ ← context vector
hₛ = encoder hidden state at position s  ·  hₜ₋₁ = previous decoder state  ·  v, W₁, W₂ = learned parameters
